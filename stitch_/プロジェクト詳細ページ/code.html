Project Overview

このプロジェクトは、アニメーション制作のスキルをアピールするために作成したポートフォリオサイトです。ターゲットユーザーは採用担当者やクリエイティブ業界の関係者とし、私のデザインと技術的な能力を直感的に理解してもらうことを目的としています。ミニマルで洗練されたデザインを採用し、コンテンツが主役となるような構成を心がけました。

My Role

UI/UXデザイン、フロントエンド開発、アニメーション実装まで、プロジェクトの全工程を一人で担当しました。ユーザー体験を最優先に考え、インタラクティブな要素を取り入れながらも、ストレスのない操作性を実現しました。
